    • Summary: Latitude vs X

    Goal: Analyze how weather changes as you approach the equator. Started with the data part first and pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

    After constructing the dataset, utilized Matplotlib in order to plot weather v. latitude. Variables considered in-scope included: cloudiness, humidity, temperature and wind speed. This website provides source data and graphics created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed therein.
